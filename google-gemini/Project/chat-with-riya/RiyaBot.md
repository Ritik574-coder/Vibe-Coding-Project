# Riya AI Chatbot

An interactive, AI-powered chat application built with React and Google's Gemini API. Riya is a virtual conversational agent designed with a predefined persona: a 24-year-old affectionate Indian woman in a romantic relationship context. The application provides a realistic messaging experience with multimedia support, voice interaction, and advanced features like smart reply suggestions and conversation analysis.

The interface is inspired by modern messaging apps (e.g., WhatsApp) and includes text, image, voice messages, and simulated voice/video calls.

## Features

- **AI-Driven Conversations**  
  Powered by Gemini 1.5 Flash with contextual history and multimodal support (text + images).

- **Persona-Based Responses**  
  Consistent character behavior using a detailed system prompt (romantic, casual Hinglish tone).

- **Multimedia Support**  
  - Send and receive text messages  
  - Image uploads with AI vision-based reactions  
  - Voice message recording and playback  
  - Text-to-speech for bot responses (with voice selection)

- **Interactive Voice/Video Calls**  
  Real-time speech recognition and synthesis for simulated live conversations (Chrome recommended).

- **Smart Features**  
  - AI-generated quick reply suggestions  
  - "Surprise Me" – spontaneous bot-initiated messages  
  - "Love Report" – AI analysis of conversation sentiment and engagement  
  - Typing indicators and read receipts

- **UI/UX**  
  - Responsive design (mobile-first)  
  - Emoji picker  
  - Custom chat background and bubble styling

## Tech Stack

- **Frontend**: React, TypeScript  
- **Styling**: Tailwind CSS (inline classes)  
- **Icons**: Lucide React  
- **AI Backend**: Google Gemini API (vision + text generation)  
- **Speech**: Web Speech API (SpeechSynthesis & SpeechRecognition)  

## Prerequisites

- Node.js (v18 or later recommended)  
- A Google Gemini API key (free tier available at [Google AI Studio])
